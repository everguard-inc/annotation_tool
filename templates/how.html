Instructions for using the Annotation Tool

1. At the beginning you need to select the project id you want to work on. Application will download images and annotations from the server, if not already downloaded. Then the project will be opened in the appropriate mode (annotation/review/correction) depending on task status on the portal.

2. You can close the application at any time, the figures will be saved and the next time you open this project - it will be opened at the same place.

3. Work duration is calculated during you are working. Application counts time intervals between key presses and mouse clicks, but no more than 10 seconds.

4. After you processed all images, press Project > Complete the project. It will send annotated data to the server and updates task status on the portal

5. The actions while annotation are performed using hotkeys. You can see them in the Help menu

6. If you need to download annotations from the server and overwrite your annotations with them, click Project > Download and overwrite annotations.

7. To go to the first image of the project click Project > Go to the first image.

8. Pay attention to the Mode section on the left side of the status bar. It shows which task you have to do.

9. Press T if you want to mark image as a trash. After that the Trash section on the status bar become red. Press T again to remove trash tag from the image.

10. You can rename BBoxes by placing the cursor over them and pressing the class hotkey. See the Help > Classes menu to see the classes, their colors, and hotkeys. In keypoint and segmentation annotation modes you can not rename figures.

11. If you want to create a figure of specific class, first press a hotkey of this class and then start creating the figure.

12. In review mode you can use point of review label to move or delete it (by pressing D). You also can rename the label by pressing another label hotkey while holding cursor over the label`s point.

13. Don't forget that you can hide figures using hotkeys while annotating or reviewing images.

14. Don't forget that you can show or hide figures names using hotkeys while annotating or reviewing images.
