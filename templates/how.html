Basic instructions for using the Annotation Tool

To get started, you need to enter the settings in the Project > Settings menu

To start annotation select project via Project > Open. Application will download images and annotations if they not already downloaded. Then the project will be opened in the appropriate mode depending on task type and status on the portal.

You can close the application at any time, your work will be saved and the next time you open this project - it will be opened at the same place.

Work duration is calculated during you are working. Application counts time intervals between key presses and mouse clicks, but no more than 10 seconds.

After you processed all images, press Project > Complete the project. It will send annotated data to the server and updates task status on the portal

Use hotkeys. You can see them in the Help menu

To go to the specific image of the project click Project > Go to image and specify image id in the field.

Pay attention to the Mode section on the left side of the status bar (bottom field on the window). It shows which task you have to do.

You can run project downloading of next project via Project > Download while annotating a previous project

Annotation

If you want to reset your work and overwrite your annotations with initial annotations, click Project > Download and overwrite annotations.

Use Ctrl+Z and Ctrl+Y for undo/redo

Place cursor under the figure and press Ctrl+C to copy it. Use Ctrl+V to paste it on another image.

Press T if you want to tag image as a trash. After that the Trash section on the status bar become red. Press T again to remove trash tag from the image.

You can rename BBoxes by placing the cursor over them and pressing the class hotkey. See the Help > Classes menu to see the classes, their colors, and hotkeys. You can rename only BBoxes and Review labels in a review mode. Keypoint groups and segmentation masks it is impossible to rename for now.

You can also use A key for renaming figures and changing active label. Place a cursor under figure, press A. You will see a widged for label selection. Move cursor to select class you want. After releasing A the figure label and active label will be changed to selected class.

In review mode you can use point of review label to move or delete it (by pressing D). You also can rename the label by pressing another label hotkey while holding cursor over the label`s point.

Hide figures using hotkeys while annotating or reviewing images if you need to see the image under them.

Don't forget that you can show or hide figures names using N while annotating or reviewing images. It is important for keypoints annotation when you need to see the names of keypoints.

You can edit visual settings in the Project > Settings > Interface
