Instructions for using the Annotation Tool

1. At the beginning you need to select the project id you want to work on. Application will download images and annotations from the server, if not already downloaded, and the project will open in the appropriate mode (annotation/review/correction) depending on task status on eg-ml.

2. You can close the application at any time, the figures will be saved and the next time you open this project - it will open in the same place.

3. Work duration calculated when you are working. Application counts time intervals between key pressews and mouse clicks, but no more than 10 seconds.

4. After you processed all images, press Project > Complete the project. It will send annotated data to the server and updates task status on eg-ml

5. The actions while annotation are performed using hotkeys. See the Help > Hotkeys menu to see the classes

6. In rare cases when you need to download annotations from the server and overwrite your annotations with them, click Project > Download and overwrite annotations.

7. Pay attention to the Mode section on the left side of the status bar. It shows which task you have to do.

8. Press T if you want to mark image as a trash. After that the Trash section on the status bar become red. Press T again to remove trash tag from the image.

9. You can rename figures by placing the cursor over them and pressing the class hotkey. See the Help > Classes menu to see the classes, their colors, and hotkeys

10. If you want to create a figure of specific class, first press a hotkey of this class and then start creating the figure.

11. In review mode you can do the same with review labels as you do with figures, using their point.

12. Don't forget that you can hide main and secondary figures using hotkeys while annotating or reviewing images. The Visibility section of the status bar can show 4 values:

  • All Visible: when both main and secondary figures are visible
  • Figures hidden: when main figures are hidden and secondary visible
  • Secondary hidden: when main figures are visible and secondary are hidden
  • All hidden: when both main and secondary figures are hidden.
