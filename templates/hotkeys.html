Hotkeys

Hotkey Action
W Save current image and open next image
Q Save current image and open previous image
D Delete a figure under cursor
T Tag/Untag the image as trash
C Replace figures on the current image with the figures from the previous image
F Fit the image into the window
H Hide/Show main figures. The main figures are those that can be edited. For example, if you are correcting a reviewed image, the bounding boxes are the main figures and the review labels are the secondary figures. If you are reviewing a project, the review labels are the main figures and the bounding boxes are the secondary figures.
J Hide/Show secondary figures
N Hide/Show figures class names
