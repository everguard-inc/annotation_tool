Hotkeys

Common

Hotkey Action
W or P Save current image and open next image
Q or O Save current image and open previous image
F Fit the image into the window
S Deteriorate image

Annotation

Hotkey Action
D Delete a figure under cursor if it is a bbox, keypoint or a reivew label. For segmentation it deletes entire mask of currently selected class
A Show class selection widget
T Tag/Untag the image as trash
Ctrl+C Copy figure under cursor
Ctrl+V Paste copied figure
Ctrl+Z Undo
Ctrl+Y Redo
E Hide/Show all figures
R Hide/Show review labels
N Hide/Show figures class names
H Hide/Show bunding bbox scale

Filtering

Hotkey Action
D or K Select/Unselect image
Z Go to the previous selected image
X Go to the next selected image
1, 2, 3, 4 Change time delay between frames change

Event Validation

Hotkey Action
1 to 9 Select answer per question
A Switch to image mode
S Switch to video mode
Z Go to the previous video frame
X Go to the next video frame
Esc Exit from comment field
